# Crosshairs Plugin for Obsidian

Adds crosshair cursor and mouse indicators for better navigation and visual tracking in Obsidian.

## Features

- **Block Cursor**: Replaces the default line cursor with a visible block cursor that blinks
- **Cursor Crosshairs**: Extends crosshair lines from the text cursor position across the entire screen
- **Mouse Crosshairs**: Shows crosshair lines following your mouse movement
- **Smart Hiding**: Mouse crosshairs automatically hide when hovering over interactive elements
- **Customizable**: Uses Obsidian's accent color variables for consistent theming

## Installation

### From Community Plugins (Recommended)
1. Open Obsidian Settings
2. Go to Community Plugins
3. Search for "Crosshairs"
4. Install and enable the plugin

### Manual Installation
1. Download the latest release from GitHub
2. Extract the files to your vault's `.obsidian/plugins/obsidian-crosshair/` folder
3. Reload Obsidian
4. Enable the plugin in Community Plugins settings

## Usage

Once enabled, the plugin automatically:
- Shows a block cursor instead of the default line cursor
- Displays crosshair extensions from your text cursor
- Shows mouse crosshairs that follow your cursor movement
- Hides mouse crosshairs when hovering over buttons, menus, and other interactive elements

No additional configuration is required.

## Compatibility

- Works with Obsidian 1.1.0 and above
- Compatible with both desktop and mobile versions
- Integrates with Obsidian's theming system
